# Link

Links are the equivalent of `a` tags, but only for textual links. Their default is to open in a new tab, but you can reset it with the `openIn` attribute.

<Canvas>
	<CmLink label="A normal link" href="https://camunda.com" />
	<CmLink
		label="Don't click me, I open in this tab"
		href="https://www.youtube.com/watch?v=oHg5SJYRHA0"
		openIn="sameTab"
	/>
</Canvas>
