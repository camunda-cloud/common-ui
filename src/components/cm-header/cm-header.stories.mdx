# Header

The header enables laying out of Logo and navigational elements with 2 slots: `left` and `right`.
Both place elements in a stacked fashion on their respective sides - in both cases from left to right.

The component makes no other assumptions about layout. Spacing between elements is left to the implementation.

	<CmHeader>
		<CmLogo slot="left" />
		<CmText slot="right">Jakob Freund</CmText>
	</CmHeader>
