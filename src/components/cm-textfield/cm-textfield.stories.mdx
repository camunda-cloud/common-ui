# Textfield

Renders a configurable textfield.

A single line textfield with placeholder, label and an icon as suffix:
<Canvas>
		<CmTextfield type="text" placeholder="Placeholder" label="Label" fieldSuffix={{type: "icon", icon: "window"}}/>
</Canvas>


A disabled multiline input with a preset value:

<Canvas>
		<CmTextfield type="multiline" rows={3} disabled value="Disabled Multiline Textfield"/>
</Canvas>
