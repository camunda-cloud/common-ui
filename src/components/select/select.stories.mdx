# Select

Selects are a thin wrapper around native selects, mostly handling the styling of the unopened select. They can be set up with an array of objects of the shape `{ label: string, value: string }` that gets passed to the `options` property.

<Preview inline>
	<CmSelect
		options={[
			{ label: 'Test', value: 'TestValue' },
			{ label: 'Another Test', value: 'AnotherTestValue' },
		]}
	></CmSelect>
</Preview>

They can be disabled of course:

<Preview inline>
	<CmSelect
		disabled
		options={[
			{ label: 'Test', value: 'TestValue' },
			{ label: 'Another Test', value: 'AnotherTestValue' },
		]}
	></CmSelect>
</Preview>
