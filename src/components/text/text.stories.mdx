# Text

This component is a simple way of getting correctly styled text into your app. It supports 3 apprearances: `normal` (the default), `subtle`, and `bold`.
It accepts its textual content in the default slot - rather than as an attribute, like most other places in this library - to enable links and other inline elements in the text.
