# Toggle

Toggles are basically [checkboxes](/?path=/docs/components-checkbox--page) and have the same API, except for the `indeterminate` state, which is not supported for toggles.

They look like this:

<Canvas>
	<CmToggle label="Check me!"></CmToggle>
	<CmToggle label="Uncheck me!" checked></CmToggle>
</Canvas>

They can be disabled of course:

<Canvas>
	<CmToggle label="Can't check me!" disabled></CmToggle>
	<CmToggle label="Can't uncheck me!" checked disabled></CmToggle>
</Canvas>
