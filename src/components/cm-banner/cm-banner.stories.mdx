# Banner

Banners are a simple way of alerting the user to a non-volatile condition. Currently two appearances are supported: `warning` and `error`.

<Canvas>
	<CmBanner appearance="warning">
		<CmText>
			This is a warning, turn it off{' '}
			<CmButton appearance="link" label="here"></CmButton>.
		</CmText>
	</CmBanner>

    <CmBanner appearance="error">
    	<CmText>
    		This is an error, fix it{' '}
    		<CmButton appearance="link" label="here"></CmButton>.
    	</CmText>
    </CmBanner>

</Canvas>
